水中运动起源于二战之后, 1950年，Virginia Hunt Newman女士开始在美国教 授婴儿及幼儿水中运动课 程，并且在后来的几十年中 影响了一大批水中运动的倡 导者，并且在60-70年代传 入其他发达国家，如、澳大 利亚、英国、日本等。优范 儿率先将美国的水中运动教 育理念引入中国，深受家长和宝宝的喜爱，优范儿积极推进并不断创新，以孩子为中心，通过国际化、先进的教育理 念和课程，尊重孩子个性，培养孩子的创造力、品格和多元文化意识，促进孩子全面发 展，使每个孩子都有一个不一样的未来。

自信胆量A 让宝贝很早的接触水会帮助宝贝抑制后期对水的恐惧。水中自由的游动，跳水及抓握 在池边可以很好的增强宝贝的自信心。
B 谨慎的宝贝学会接受风险，容易过度兴奋的则会变得比较谨慎。
C 为宝贝提供一个丰富且富有刺激性的环境让宝贝在探索中变得更勇敢。

协调平衡A 在水中能很好的增强协调和平衡能力。
B 让宝贝自己在浅区行走，锻炼平衡感。
C 让宝贝趴在浮棍上锻炼协调和平衡。
D 让宝贝坐、趴、站在漂浮垫上。

幸福感A 孩子在温暖的水中会很放松，心情愉悦。
B 水中运动对孩子消化系统有很好的帮助。
C 同时促进孩子的睡眠和饮食质量。

充实每天A 每周的水中运动课能充实孩子每天的生活。
B 可以给孩子提供有趣的社交环境。
C 同龄段社交环境可以让孩子成长更快。

学习能力A 不断的重复性信号词会增强宝贝的智力和理解能力。
B 德国研究发现水中运动的宝贝有更好的动作发展，社交能力和智力发展。
C 让宝宝较为容易地学会分享，排队，并接受失望。
D 常态性的课程有助于建立互动互助。

增强力量A 水中的浮力和阻力更能练习宝贝的肌肉力量，水中比陆地有更明显的效果。
B 芬兰研究表明很好的肌肉控制能力对后期宝贝爬行和行走也大有益处。
C 使宝贝强化身体，有助于肌肉骨骼发展。
